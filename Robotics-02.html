Testing Invent! on your WiFi
Putting it in setup mode

You've been sent an Invent! main board so you can test if it connects to your WiFi network and can be programmed over WiFi.

You'll first need to configure it with your WiFi details. For this, you first need to put Invent! in setup mode. You'll be powering the board via USB, so plug the cable in between the Invent! board and a power source such as a USB socket on a computer.

Hold down the button shown, whilst you turn on the robot by pushing the slide switch to the 'On' position.

Your robot should say the words 'WiFi', then 'Access', and then 'Setup'. As soon as you've heard it say 'Setup', release the button.

Connecting to the access point

Your robot will now act as an access point, and be broadcasting a WiFi name 'InventXXXXXX' where the characters at the end will be the same as the bold characters on the label in the picture. You will need to connect your computer to this access point before proceeding.

The config screen

Start a web browser and enter the address '192.168.4.1' into the address bar at the very top of the screen as shown.

If you are connected to the Invent! WiFi and have typed in the correct address you should get a screen that looks like this.

Entering your details

Enter your WiFi name (SSID) and password and press the 'Save' button. You should get a message asking you to restart the robot. Do this by turning it off and on again (without holding any buttons in this time).

N.B. Be careful you have typed the details correctly, paying close attention to any uppercase/lowercase characters and special symbols.

If you are successful, then the LED next to the button should briefly go red and then turn green. This indicates it is connected. If it stays red, you have not entered the correct SSID or password - you will need to repeat these steps again.

Your Invent! board remember your WiFi details and reconnect to your WiFi each time it is turned on.

Loading Invent Blocks

Go to the website http://code.inventblocks.com/ui/ and click on the WiFi icon shown.

Entering the IP address

You will need to connect to the Invent! board using it's IP address. This is a series of four numbers with dots in between them which is used on all computer networks. When your board connects to WiFi it will get it's own IP address.

To get this address, you'll need to turn the board off and on again, whilst holding down the button as before, but this time you will release the button when the light turns red and the robot says 'Wifi'. After a few seconds, the board will connect to the Wifi network as before (and the light should go green), but this time, it will speak its IP address and say something like '192.168.1.1'. Type this in the 'Address' box and click the 'Network' button above it to connect.

Success!

After a few seconds the connect icon shown should go solid blue and look like the cables are connected. If it looks like the screen shown, your robot has connected. If it stays black and looks disconnected, there has been a problem. In this case, you should:

  • Check your board has connected to the WiFi (the green light is on)
  • Check your computer is connected to the same WiFi network
  • Check you have entered the board's IP address correctly in the previous step.
Test Code

Now let's check that we can program our Invent! board. Go to the 'Invent' group of blocks at the top left, and drag the 'LED' block onto your workspace as shown.

Press the 'Play' button on the top right. You should see a LED on your board go red. This means that everything is now tested - your board is connected to your WiFi, and you have successfully programmed the board to turn a LED on. You might like to try changing the color of the LED and press play again.

You don't have a motor module to make the robot move but for a bit of fun you could see if you can find the sound blocks to play a tune like 'Mario Brothers' on your Invent! board!
